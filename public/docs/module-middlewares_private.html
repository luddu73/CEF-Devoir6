(static) checkJWT(req, res, next) → {response}

Vérifie l'existance et la validité du token JWT dans l'header des requêtes - Si le token est valide, il est décodé et rattaché à req.decoded - On créer un nouveau token renvoyé dans l'header "Authorization" pour continuer la session
